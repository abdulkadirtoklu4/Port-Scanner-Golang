# Linux Install Steps:

```shell

    apt-get update -y && apt-get upgrade -y
    wget https://golang.org/dl/go1.17.2.linux-amd64.tar.gz
    sudo tar -C /usr/local -xzf go1.17.2.linux-amd64.tar.gz
    echo 'export PATH=$PATH:/usr/local/go/bin' >> ~/.profile
    source profile
    go version

    // run;
    go run <filename>
```
